# InstaKart-front
This is the front end for a supermarket/grocery eCommerce application.

## Pages

So far, this application includes:

```
Pages:
- home page
- sign up page
- sign in page

Admin Pages:
- Main Dashboard
- List of all products (active and inactive products)
```

## Features

```
Features:
- sign up
- sign out
- log in with email and password
- log in with Google
- email confirmation
  Admin Features:
  - create products
    - title
    - description
    - multiple images
    - price
    - price per unit
    - inventory
    - suggested inventory
    - L W H dimensions
    - weight
    - taxable
    - active/non active
    - available for shipping or pickup
  - update products
  - delete products
- ...much more
```
